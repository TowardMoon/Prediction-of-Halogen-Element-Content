# Prediction-of-Halogen-Element-Content
We divided all the data into three major categories, each with four variables, generating a total of twelve models.
The steps of filtering out data with too many missing values and outliers are completed in Excel.
We carried out data preprocessing, model construction, and application in sequence.
Data preprocessing steps obtain the features for each predictive model, and grid search obtains the model parameters. We import the above content into the model-building section to get our final model.
